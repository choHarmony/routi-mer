## ⏲ 루티머 (Routi-mer) ⏲</br>  
  
</div>    

### 🗂 소개    
> 내가 원하는 타이머를,  
> 내가 원하는 횟수만큼,  
> 내가 원하는 순서대로!
>   
> 지금 여러 개의 타이머들을 순차적으로 실행할 수 있는 `루티머`에서,  
> 내 마음대로 타이머 루틴을 만들어보세요!
</br>

### ✅ 이런 분들에게 유용해요  
```
- 동시에 여러 개의 타이머를 실행하는 게 아니라, 여러 타이머를 순서대로 실행시키고 싶은 사용자  
- 용도에 따라 다양한 타이머를 사용하는 사용자
```
  </br>
